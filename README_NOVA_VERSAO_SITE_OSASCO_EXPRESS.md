# Nova versão do Site Oficial Osasco Express

Versão criada conforme Documento Mestre Profissional.

## Direção estratégica

- +500 mil entregas realizadas.
- +8 anos de experiência.
- Tecnologia própria e agenda digital de disponibilidade de entregadores parceiros.
- Suporte geral das 10h às 22h.
- Para restaurantes, acompanhamento durante o turno operacional combinado.
- Foco principal: entregas, motoboy, logística local, última milha e base operacional de entregadores parceiros.
- Apoio operacional complementar apresentado com cuidado, sem tirar o foco principal de entrega.
